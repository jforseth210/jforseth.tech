How To:

Recieve prayer requests:

  1. Type in your email address where it says "Or be notified when someone requests prayer:"
  2. If you were given a code, type it in the box that say "PARISH". Otherwise leave this blank.
  3. Click submit
  4. Check your email. There should be a message from support@jforseth.tech.
  5. Open the message. Click the verify button at the buttom.
  6. Congrats. You will now recieve prayer requests.

Submit a prayer request:

  1. Where it says "nickname" type your name or a nickname if you perfer anonymity.
  2. Choose the group you'd like to send the request to. Selecting Public will send it to all groups. Please do not send the prayer request to Testing.
  3. Enter your prayer request in the text box.
  4. Click submit.
  5. Please wait for the page to load, your request is being sent.

Type your name or a nickname if you perfer anonymity. Type out your prayer request. In the current verision, select an amount of time you'd like us to pray for you. In an upcoming version, select the group you'd like the request to be sent to. When you click submit, this request is sent as an email to the community.
